📋 Command Cheatsheet

Your survival guide. Keep this handy — these are all the commands you need today.

🧭 Moving Around

pwdWhere am I?
lsWhat's here?
ls -aShow everything — even hidden!
cd folderGo into a folder
cd ..Go back one folder
cd ~Go home

📄 Files

cat fileRead a file
touch fileCreate a new file
mv old newRename or move
cp file copyCopy a file
rm fileDelete a file

📁 Folders & Search

mkdir nameCreate a folder
grep "word" fileSearch in a file
wc -l fileCount lines

📝 Scripts & More

nano fileOpen text editor
bash fileRun a script
echo "hi"Print text
clearClean the screen
historySee past commands
💡
When you type ls, notice the colors! Names in blue are folders you can enter with cd. Names in white are files you can read with cat.

🎯 Pro Tips

Shortcuts that make you faster than everyone else.

Tab ↹

Auto-completes file and folder names. Your best friend.

↑ Arrow

Brings back your last command. Press it again for older ones.

cd ~

Lost? Go home instantly. Always works.

clear

Screen too messy? Clean it up in one command.

Ctrl + C

Something won't stop? Force quit any running command.

Ctrl + O → Enter → Ctrl + X

The nano save-and-exit combo. Memorize it!
