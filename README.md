# 🐧 Linux Adventure Workshop

## You Don't Just Use Linux — You BUILD With It!

Welcome to the **Linux Adventure Workshop**! In this session, you'll explore the Linux terminal, discover hidden files, build things with commands, and even write your first program.

No clicking. No menus. Just you and the terminal. Let's go.

---

## 🎯 What You'll Learn

- How to navigate the terminal like a pro
- How to create, move, and find files using commands
- How to write and run your first script (your first program!)
- That Linux is a **builder's tool** — not just something you install

---

## 💻 What You Need

- A Windows laptop with **WSL Ubuntu** installed
- That's it! Everything else is built in

To open your terminal:
1. Press `Windows key`
2. Type `Ubuntu`
3. Press Enter

You're in! 🎉

---

## 👩‍🏫 For Instructors

### Before the Workshop

1. Open Ubuntu on every laptop and run the setup script:
   ```bash
   bash setup.sh
   ```
2. Optionally, install fun tools (see `bonus/setup-fun.sh`)

### During the Workshop

- Students work in groups of **1–2 per laptop**
- One person types, others help — rotate every few minutes
- Follow the timeline in [`instructor-guide.md`](instructor-guide.md)
- Print or display the [`cheatsheet.md`](cheatsheet.md)

### Group Tips

- Encourage students to figure things out before giving answers
- Celebrate mistakes — they're part of learning
- Keep energy high: this is a playground, not a lecture
