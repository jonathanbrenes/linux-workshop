What is Open Source?

🔒

Closed Source

Someone builds software and hides the recipe. You can use it, but you can't see how it works or change it.

Examples: Windows, macOS, Instagram

🔓

Open Source

The recipe is shared with the world. Anyone can read it, improve it, and share it — for free.

Examples: Linux, Firefox, Android, VLC

Linux is open source. Thousands of people around the world build it together.

Even Microsoft uses Linux

Yes — the company that makes Windows.

☁️
Azure Linux 4.0 — Microsoft just launched their own Linux distro to power Azure, the world's second-largest cloud.
🪟
WSL — Windows Subsystem for Linux lets you run Linux inside Windows. That's what we're using today!
🐙
GitHub — the world's largest code platform (owned by Microsoft) runs on Linux servers.
🤖
AI infrastructure — Microsoft trains and runs its AI models on Linux. Copilot, Bing Chat, and Azure AI all run on Linux servers.
