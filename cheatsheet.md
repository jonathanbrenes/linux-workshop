# 📋 Linux Command Cheatsheet

Print this out or keep it open on the side!

---

## 🧭 Moving Around

| Command | What It Does | Example |
|---------|-------------|---------|
| `pwd` | **Where am I?** Shows your current folder | `pwd` |
| `ls` | **What's here?** Lists files and folders | `ls` |
| `ls -a` | **Show EVERYTHING** — even hidden files! | `ls -a` |
| `cd foldername` | **Go into** a folder | `cd cave` |
| `cd ..` | **Go back** one folder | `cd ..` |
| `cd ~` | **Go home** — back to your home folder | `cd ~` |

---

## 📄 Working With Files

| Command | What It Does | Example |
|---------|-------------|---------|
| `cat filename` | **Read** a file | `cat clue1.txt` |
| `touch filename` | **Create** a new empty file | `touch arm.txt` |
| `mv old new` | **Rename** or **move** a file | `mv arm.txt robot-arm.txt` |
| `cp file copy` | **Copy** a file | `cp arm.txt arm-backup.txt` |
| `rm filename` | **Delete** a file (careful!) | `rm junk.txt` |

---

## 📁 Working With Folders

| Command | What It Does | Example |
|---------|-------------|---------|
| `mkdir name` | **Create** a new folder | `mkdir my-robots` |
| `rmdir name` | **Delete** an empty folder | `rmdir old-folder` |

---

## 🔍 Finding Things

| Command | What It Does | Example |
|---------|-------------|---------|
| `grep "word" file` | **Search** for a word inside a file | `grep "cookie" witness1.txt` |
| `grep -r "word" folder` | **Search everywhere** in a folder | `grep -r "cookie" evidence/` |
| `wc -l filename` | **Count** lines in a file | `wc -l parts.txt` |

---

## 📝 Writing Scripts

| Command | What It Does | Example |
|---------|-------------|---------|
| `nano filename` | **Open** a simple text editor | `nano hello.sh` |
| `bash filename` | **Run** a script | `bash hello.sh` |
| `echo "text"` | **Print** text to the screen | `echo "Hello!"` |

---

## 🧹 Other Useful Stuff

| Command | What It Does |
|---------|-------------|
| `clear` | **Clean** the screen |
| `history` | **See** all commands you typed |
| `man command` | **Read the manual** for any command |
| `↑ arrow` | **Repeat** your last command |
| `Tab` | **Auto-complete** file and folder names |

---

## 🎯 Pro Tips

- **Tab key** is your best friend — it auto-completes names so you don't have to type everything
- **Up arrow** brings back your last command
- If you get lost, type `cd ~` to go home
- If the screen is messy, type `clear`
- File names with spaces need quotes: `cat "my file.txt"`
